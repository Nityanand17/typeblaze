Features

Live Scoreboard: Monitor your rank and progress throughout each match.

Custom Game Rooms: Create personalized game sessions with adjustable settings.

Cross-Platform Compatibility: Fully responsive design for seamless gameplay on desktop and mobile.

Getting Started

Follow these steps to set up and run TypeBlaze on your local machine.

Prerequisites

Ensure you have the following installed:

Node.js (v20 or newer)

npm (or Yarn, pnpm, or bun)

Git (for cloning the repository):
git clone https://github.com/yourusername/typeblaze.git

Navigate to the project directory and install dependencies:
cd typeblaze
npm install
npm run dev

Open your browser and visit http://localhost:3000 to start playing.

How to Play

Join or Create a Room: Enter an existing game room or set up your own with custom rules.

Compete in Typing Challenges: Type the given text accurately and as quickly as possible.

Track Your Progress: View your typing speed and accuracy on the live scoreboard.

Win the Game: The player with the highest WPM and accuracy at the end of the match wins!


Deployment

TypeBlaze’s front-end is deployed on Vercel. We are currently exploring affordable solutions for hosting the back-end, which consists of an Express and Socket.io server. Authentication is handled through Next.js API routes.

Stay tuned for further updates and improvements!
